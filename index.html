⭐ Recomendado · Gratuito
Cómo obtener tu API Key de Gemini
1
Entrá a aistudio.google.com/apikey con tu cuenta Google.
2
Hacé clic en Create API key → seleccioná o creá un proyecto.
3
Copiá la key — comienza con AIza... — y pegala abajo.
4
Plan gratuito: 250 requests/día con Gemini 2.5 Flash. Sin tarjeta.
Claude Sonnet · Anthropic
Cómo obtener tu API Key de Claude
1
Entrá a console.anthropic.com/keys con tu cuenta Anthropic.
2
Hacé clic en Create Key → asignale un nombre descriptivo.
3
Copiá la key — comienza con sk-ant-... — y pegala abajo.
4
Claude requiere créditos prepagos. Tiene costo pero alta calidad clínica.
GPT-4o · OpenAI
Cómo obtener tu API Key de OpenAI
1
Entrá a platform.openai.com/api-keys con tu cuenta OpenAI.
2
Hacé clic en Create new secret key → asignale un nombre.
3
Copiá la key — comienza con sk-... — y pegala abajo. Se muestra una sola vez.
4
GPT-4o requiere créditos prepagos. Útil para informes de obras sociales.
Las keys se guardan solo en tu navegador
